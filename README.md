 Problem Statement 
 Farmers often face challenges like unpredictable weather , crop selection issues, irrigation management and market price fluctuations.
 This system solves this problem using AI agents that work together to provide intelligent farming solutions or suggestions.

 System Architecture
 This project is built using multiagent architecture.
 It includes four main agents: weather agent, crop agent, irrigation agent and market price agent.
 All agents are coordinated using root agent and mcp style architecture. .env file is used to store keys and security module for authentication.

 Solution to problem statement 
 multiagent system contains various agents such as weather agent, it provides weather based farming suggestions to help farmers plan their activities.
 crop agent suggests suitable crops based on climate and seasonal conditions.
 irrigation agent ensures efficient irrigation and better farming results.
 market agent helps farmers decide best time to sell their crops for maximum profit.
